# 📌 Project Overview

This project uses **Machine Learning Regression** techniques to estimate the selling price of a used car based on different features such as manufacturing year, fuel type, transmission, kilometers driven, and more.

The application is built using **Python**, **Scikit-Learn**, and **Streamlit**, providing users with an easy-to-use interface for instant price prediction.

---

# 🎯 Objective

- Predict the selling price of a used car accurately.
- Perform data preprocessing and visualization.
- Train a Machine Learning model.
- Deploy an interactive web application using Streamlit.

---

# 📂 Dataset

**Source:** Kaggle Used Car Dataset

The dataset contains information such as:

- Manufacturing Year
- Present Price
- Kilometers Driven
- Fuel Type
- Seller Type
- Transmission
- Owner Count

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/your-repository.git
```

Move into the project directory

```bash
cd Car_Price_Prediction
```

Install the required libraries

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Project

### Train the Model

```bash
python main.py
```

### Start the Streamlit App

```bash
streamlit run app.py
```
