# Iris Flower Classification

## Objective

Build a Machine Learning model to classify Iris flowers into:

- Iris-setosa
- Iris-versicolor
- Iris-virginica

using flower measurements.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Dataset

Iris Dataset from Kaggle.

## Model Used

Random Forest Classifier

## Accuracy

Around 95% to 100%

## How to Run

pip install -r requirements.txt

python main.py
