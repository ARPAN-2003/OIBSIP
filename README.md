# 📌 Repository Overview

This repository contains multiple Machine Learning projects completed as part of the **AICTE Oasis Infobyte Data Science Internship**.

Each project includes:

- ✅ Data Preprocessing
- ✅ Exploratory Data Analysis (EDA)
- ✅ Data Visualization
- ✅ Machine Learning Model
- ✅ Model Evaluation
- ✅ Prediction Module
- ✅ Streamlit Web Application
- ✅ Project Documentation

# ▶️ Getting Started

Clone the repository

```bash
git clone https://github.com/ARPAN-2003/OIBSIP.git
```

Go inside the project

```bash
cd OIBSIP
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run any project

```bash
python main.py
```

or

```bash
streamlit run app.py
```
