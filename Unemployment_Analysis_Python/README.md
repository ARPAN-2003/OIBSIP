# 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on India's unemployment dataset to identify patterns, trends, and relationships between unemployment rate, employment count, and labour participation rate.

The project also generates visual reports and provides meaningful insights through graphs and statistical analysis.

---

# 🎯 Objectives

- Analyze unemployment trends across different states
- Compare labour participation rates
- Study regional unemployment patterns
- Generate visual reports
- Understand correlations between employment-related factors

---

# ✨ Features

✅ State-wise unemployment analysis

✅ Region-wise analysis

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/ARPAN-2003/OIBSIP.git
```

Go to the project folder

```bash
cd OIBSIP/UNEMPLOYMENT_ANALYSIS_PYTHON
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python main.py
```

---

# 💻 Streamlit Dashboard
